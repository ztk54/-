# 一维数组  
1. 在输出数组元素时候，一般使用for循环输出，不过字符数组可以使用其特性进行输出。
> [!tips]
> ```c
> 	char arr[10]={'a','b','c','d'};
> 	 printf("%s",arr);
>```

这样子就能输出`abcd`，但是对于其他整形，浮点型的数组不能进行一次性输出，必须使用for循环进输出。
2. 使用sizeof进行计算数组元素个数。
   ```c
	   int arr[20]={0};
	   int sz=sizeof(arr)/sizeof(arr[0]);
   ```  
   这样子即使arr数组发生变化，在后续输出或者输入数组元素时候就不用频繁变化for循环中的`for(int i=0;i<sz;i++)`中sz的数值了。
# 二维数组  
1. 初始化  
	1. 完全初始化，将二维数组的每个元素都赋予准确的数值。
	2. 不完全初始化，只初始化数组的部分元素。
	3. 按行初始化， `int arr[3][5]={{1，2，3}，{5，6，7}，{5，9，8}}`，这样子就可以对每一行进行赋值了。
2. 变长数组在vs2026里面不支持，变长数组不能够进行初始化。
> [!tips]
> vs2026里面可以通过更换编译器的方式来使用变长数组。在vs2026installer里面修改添加clang编译器，如何在文件内右击鼠标，在属性里面可以修改编译器。
