二分查找是用于实现**升序数组**查询某一元素的策略，代码如下：
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main()
{
	int arr[] = { 1,3,4,6,8,12,15,16,18,19,20 };
	int left = 0, right = sizeof(arr) / sizeof(arr[0]) - 1;
	int flag = 0;
	int number = 1;
	printf("请您输入您想查找的数据\n");
	scanf("%d", &number);
	while (left <= right)
	{
		int mid = (left + right) / 2;//int mid=left+(right-left)/2;
			if (arr[mid] > number)
			{
				right = mid - 1;
			}
			else if (arr[mid] < number)
			{
				left = mid + 1;
			}
			else
			{
				printf("找到了，下标为：%d\n", mid);
				flag = 1;
				break;
			}
	}
	if (flag == 0)
	{
		printf("找不到这个数字\n");
	}
	return 0;
}
```
>[!tips]
>这里我犯了两个很严重的错误，导致代码出现了一些问题。
>1. 首先我忘记了在循环内部更新mid的大小，导致mid数值不会发生更新，因此没有实现二分查找的功能实现。
>2. 其次我为数组开创了40个int元素的空间，但是实际上只使用了一小部分，导致在sizeof那里计算right的时候出现了一些问题。

>[!warning]
>在这里计算mid的时候，可能会发生左值加右值大于数据类型的范围的情况，因此我们可以使用`mid=left+(right-left)/2`来计算mid的大小，这样子就避免了数据溢出的问题。
